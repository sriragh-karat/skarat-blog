# Skarat Blog

A personal blog built with [11ty](https://www.11ty.dev/), a simpler static site generator.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm

### Installation

```bash
git clone git@github.com:sriragh-karat/skarat-blog.git
cd skarat-blog
npm install
```

### Development

To run the blog locally:

```bash
npm start
```

The site will be available at `http://localhost:8080`

### Build

To generate a production build:

```bash
npm run build
```

## Project Structure

- `src/` - Source files (content, layouts, assets)
- `_site/` - Generated static site (output)
- `.eleventy.js` - 11ty configuration

## Deployment

This site can be deployed to any static hosting service (GitHub Pages, Netlify, Vercel, etc).
